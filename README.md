# TP_Final_SO_Sockets
Proyecto de Sockets para TP Final de SO UNaHur 2025C2


## Terminal integrada 1 - Servidor:
- Correr el servidor en consola:
dotnet run --project .\Servidor\Sockets.csproj

## Terminal integrada 2 - Cliente:
- Correr el cliente en consola: <Reemplazar 127.0.0.1 por la IP del Servidor>
dotnet run --project .\Cliente\Sockets.csproj -- 127.0.0.1
